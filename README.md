## Getting Started

1. Make sure you have Ruby 2.2.2
2. `bundle install`
3. `cp .env.example .env`
4. `bin/rake db:create db:migrate`
5. `bin/rake db:test:prepare`
6. `bin/rspec`
7. `bin/rails s`
